## Description

> This project lets users play as a blob traversing an obstacle course using keyboard inputs.
> It explores how emotion can be represented by mood states.

---

## Interaction Instructions

> Press `W` or up arrow to jump up, press `A` or left arrow to move left, and press `D` or right arrow to move right.
